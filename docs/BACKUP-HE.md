# גיבוי נתונים — ARTIST CRM

## איך הנתונים נשמרים

| שכבה | תיאור |
|------|--------|
| **שרת (Neon)** | כל יצירה / עריכה / מחיקה נשמרת מיד במסד PostgreSQL בענן |
| **גיבוי JSON** | הורדה ידנית של כל המאגר לקובץ מקומי |
| **ייצוא CSV** | ייצוא לפי סינון נוכחי (Excel / Google Sheets) |

אין שמירה מקומית בדפדפן לנתוני אומנים — המקור האמיתי הוא תמיד השרת.

## הורדת גיבוי

1. לחץ **גיבוי** (בכותרת במחשב, או בסרגל התחתון בנייד)
2. יורד קובץ `artist-backup-YYYY-MM-DD.json`
3. הקובץ כולל: תאריך, סטטיסטיקות, ורשימת כל האומנים

API: `GET /api/backup`

## תזכורת שבועית

אם לא הורדת גיבוי מעל 7 ימים — מוצגת תזכורת עם כפתור **גיבוי עכשיו**.

תאריך הגיבוי האחרון נשמר ב-`localStorage` תחת המפתח `artist-last-backup`.

## המלצות

- הורד גיבוי JSON **לפחות פעם בשבוע**
- שמור עותק בענן (Google Drive, OneDrive) או כונן חיצוני
- Neon (ספק מסד הנתונים) מבצע גיבויים תשתיתיים — הגיבוי המקומי הוא שכבת ביטחון נוספת

## שחזור

שחזור מלא מקובץ JSON דורש הרצת סקריפט ייבוא (פנה למפתח).  
לשינויים יומיומיים — השתמש בממשק האפליקציה בלבד.
